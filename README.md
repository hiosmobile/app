# HiOSUnified by The Highland Cafe™

Welcome to the source code for our latest project, HiOSUnified 3!!

This version aims to replace the Java wrapper for HiOSMobile, and provide a desktop version for Windows and MacOS users!

It is built with Tauri, React, and Vite, and uses code from https://github.com/hiosmobile/hiosmobile.github.io to provide the wrapper UI.

Try it here, it is currently in beta: https://github.com/hiosmobile/app/releases/.
